# BoxManager

Repository caricato da locale — progetto ASP.NET Core (MVP).

## Requisiti
- .NET 8 SDK

## Esecuzione locale
1. Apri la cartella `BoxManager` in Visual Studio o VS Code.
2. Costruisci e avvia:

```powershell
dotnet build
dotnet run
```
